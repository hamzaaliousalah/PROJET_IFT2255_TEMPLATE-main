# Flux principaux

## Objectif

Les flux décrivent les interactions globales entre les acteurs et le système pour les cas d’utilisation majeurs.

## Flux d’activités principaux

1. **Recherche d’un cours**
   - Étudiant → saisit un code/nom → API Planifium → vérification prérequis → affichage résultats.

2. **Comparaison de cours**
   - Étudiant → sélectionne 2 cours → récupération données officielles, académiques, avis → affichage tableau comparatif.

3. **Consulter la fiche d’un cours**
   - Étudiant → clique sur un cours → système récupère données officielles et avis → affiche fiche détaillée.

4. **Gestion du profil**
   - Étudiant → ouvre son profil → modifie informations → DB mise à jour → confirmation immédiate.

5. **Recommandations personnalisées**
   - Étudiant → consulte recommandations → système analyse profil académique → propose liste de cours.
